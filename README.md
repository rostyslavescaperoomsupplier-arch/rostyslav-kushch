# Rostyslav Kushch — Personal Portfolio

Single-file, self-contained portfolio site for **Rostyslav Kushch** — Hardware & Software Developer, President of the WIWAV scientific circle (ZUT, Szczecin).

**Live:** https://rostyslavescaperoomsupplier-arch.github.io/rostyslav-kushch/

## Features
- One `index.html`, no build step, no dependencies (works offline).
- Multilingual: **EN / UA / PL / RU** with a language switcher (auto-detects browser language, remembers choice).
- Sections: About · Skills · full STEP IT Academy "Software Development" program · Projects · Journey · Experience · Diplomas · Contact.
- Dark "engineering" theme, responsive, animated on scroll.

## Add your photo
Drop a square image named **`photo.jpg`** next to `index.html`, commit and push:
```
git add photo.jpg && git commit -m "Add photo" && git push
```
Until then the site shows a stylish "RK" monogram avatar.
